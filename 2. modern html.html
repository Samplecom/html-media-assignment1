2.Html entities & - Ampersand Entity: Represents the ampersand character (&). This entity is essential for escaping the actual ampersand when it is part of text content or attributes. < - Less-Than Entity: Represents the less-than symbol (<). It is used to display a less-than sign within the content of an HTML document. > - Greater-Than Entity: Represents the greater-than symbol (>). It is used to display a greater-than sign within the content of an HTML document. " - Double-Quote Entity: Represents the double-quote character ("). This entity is used to display double quotation marks within attributes that are wrapped in double quotes.
